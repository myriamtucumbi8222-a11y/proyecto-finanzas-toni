# Agente de Valoración

## Objetivo

Estimar el valor de Industrias Lácteas Toni S.A.

## Responsabilidades

* Aplicar el método de Flujo de Caja Descontado.
* Aplicar un método de contraste.
* Estimar el valor terminal.
* Comparar el valor estimado con el valor observado.

## Entradas

* Flujo de caja.
* WACC.
* Información financiera.

## Productos esperados

* Valor empresarial.
* Informe de valoración.

## Prompt sugerido para Codex

"Estima el valor de Industrias Lácteas Toni S.A. mediante flujo de caja descontado y un método de contraste."

# Agente de Datos

## Objetivo

Recolectar, limpiar, organizar y validar la información económica y financiera utilizada en el proyecto.

## Responsabilidades

* Recopilar datos de fuentes oficiales.
* Verificar la calidad de los datos.
* Organizar la base de datos.
* Elaborar el diccionario de datos.
* Documentar las fuentes utilizadas.

## Entradas

* Estados financieros.
* Indicadores económicos.
* Bases de datos oficiales.

## Productos esperados

* Base de datos organizada.
* Diccionario de variables.
* Datos listos para el análisis.

## Prompt sugerido para Codex

"Recolecta y valida los datos financieros y económicos de Industrias Lácteas Toni S.A., documentando las fuentes oficiales utilizadas."

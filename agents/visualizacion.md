# Agente de Visualización

## Objetivo

Diseñar el dashboard interactivo para presentar los resultados del proyecto.

## Responsabilidades

* Diseñar gráficos.
* Organizar indicadores.
* Presentar resultados financieros.
* Preparar la visualización para Vercel.

## Entradas

* Resultados finales del proyecto.

## Productos esperados

* Dashboard interactivo.
* Visualizaciones.
* Presentación de resultados.

## Prompt sugerido para Codex

"Diseña un dashboard interactivo que presente los indicadores económicos y financieros de Industrias Lácteas Toni S.A. de forma clara y comprensible."

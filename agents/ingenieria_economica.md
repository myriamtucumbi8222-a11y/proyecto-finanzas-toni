# Agente de Ingeniería Económica

## Objetivo

Evaluar la rentabilidad económica de las inversiones realizadas por la empresa.

## Responsabilidades

* Elaborar el flujo de caja.
* Calcular VAN.
* Calcular TIR.
* Calcular Relación Beneficio-Costo.
* Calcular el período de recuperación.
* Realizar análisis de sensibilidad.

## Entradas

* Inversión inicial.
* Flujo de caja.
* Tasa de descuento.

## Productos esperados

* Evaluación de inversiones.
* Indicadores de ingeniería económica.

## Prompt sugerido para Codex

"Evalúa la viabilidad financiera de las inversiones de Industrias Lácteas Toni S.A. mediante VAN, TIR, Beneficio-Costo y análisis de sensibilidad."

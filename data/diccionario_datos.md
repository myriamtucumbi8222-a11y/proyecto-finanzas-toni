# Diccionario de datos

## Proyecto

Análisis económico, financiero y de ingeniería económica de Industrias Lácteas Toni S.A.

## Descripción

Este documento contiene la descripción de las variables utilizadas para el análisis económico y financiero del proyecto.

---

## Variables financieras

| Variable      | Descripción                                                      | Fuente              |
| ------------- | ---------------------------------------------------------------- | ------------------- |
| Ingresos      | Valor obtenido por ventas de productos y servicios de la empresa | Estados financieros |
| Costos        | Recursos utilizados para la producción                           | Estados financieros |
| Gastos        | Gastos administrativos y operativos                              | Estados financieros |
| Activos       | Recursos económicos propiedad de la empresa                      | Estados financieros |
| Pasivos       | Obligaciones financieras y comerciales                           | Estados financieros |
| Patrimonio    | Recursos aportados por los propietarios                          | Estados financieros |
| Utilidad neta | Resultado final después de costos, gastos e impuestos            | Estados financieros |

---

## Indicadores financieros

| Variable           | Descripción                                                     |
| ------------------ | --------------------------------------------------------------- |
| Liquidez corriente | Capacidad de la empresa para cubrir obligaciones de corto plazo |
| Endeudamiento      | Nivel de participación de deuda en la estructura financiera     |
| ROA                | Rentabilidad generada por los activos                           |
| ROE                | Rentabilidad obtenida por los accionistas                       |
| WACC               | Costo promedio ponderado del capital                            |

---

## Variables de ingeniería económica

| Variable                | Descripción                                             |
| ----------------------- | ------------------------------------------------------- |
| Inversión inicial       | Recursos destinados al proyecto de inversión            |
| Flujo de caja           | Entradas y salidas de efectivo proyectadas              |
| VAN                     | Valor actual de los flujos futuros descontados          |
| TIR                     | Tasa que iguala el valor presente de ingresos y egresos |
| Beneficio-costo         | Relación entre beneficios y costos del proyecto         |
| Periodo de recuperación | Tiempo necesario para recuperar la inversión            |

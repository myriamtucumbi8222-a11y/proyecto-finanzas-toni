# Metodología del proyecto

## Proyecto

Análisis económico, financiero y de ingeniería económica de Industrias Lácteas Toni S.A.

## Enfoque metodológico

El proyecto utiliza un enfoque cuantitativo basado en el análisis financiero, evaluación de inversiones y valoración empresarial.

## Etapas desarrolladas

### 1. Recolección de información

Se recopilan datos económicos y financieros provenientes de fuentes verificables.

### 2. Procesamiento de datos

Los datos son organizados, limpiados y preparados para el análisis.

### 3. Análisis financiero

Se calculan indicadores de liquidez, endeudamiento y rentabilidad.

### 4. Evaluación económica

Se aplican herramientas como:

* VAN.
* TIR.
* Beneficio-costo.
* Periodo de recuperación.

### 5. Análisis de riesgos

Se elaboran escenarios optimista, base y pesimista.

### 6. Recomendación

Los resultados obtenidos permiten formular una recomendación financiera sustentada.

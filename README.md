# Análisis Económico, Financiero y de Ingeniería Económica

## Industrias Lácteas Toni S.A.

---

## Descripción del proyecto

Este proyecto desarrolla un análisis económico, financiero y de ingeniería económica aplicado a Industrias Lácteas Toni S.A.

El objetivo es evaluar decisiones de inversión, financiamiento y generación de valor mediante herramientas financieras y un sistema multiagente apoyado por inteligencia artificial.

---

## Objetivo general

Analizar la viabilidad económica y financiera de las inversiones realizadas por Industrias Lácteas Toni S.A., utilizando indicadores financieros, evaluación de proyectos, valoración empresarial y análisis de riesgos.
